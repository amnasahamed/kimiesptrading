Required Parts

  • ESP8266 - NodeMCU v3 or Wemos D1 Mini
  • OLED Display - 1.3" 128x64 SSD1306/SH1106 I2C (4-pin)
  • Push Button - Momentary switch
  • Buzzer - Active piezo buzzer (optional)
  • Jumper Wires - Female-to-female Dupont cables

Note: 1.3" OLEDs may use address 0x3C or 0x3D. If display is blank, try changing OLED_ADDR in the code.

Wiring Diagram

1.3" OLED Display (I2C)

GND → GND
VCC → 3V3
SCL → D1 (GPIO5)
SDA → D2 (GPIO4)

Navigation Button

D6 (GPIO12) → Button → GND

*Use D6 (GPIO12) NOT D3 - D3 is the FLASH button used for programming mode

Alert Buzzer (Optional)

D5 (GPIO14) → Buzzer+ → Buzzer- → GND

Troubleshooting

Display stays blank?

Try changing OLED_ADDR to 0x3D in the code. Some 1.3" OLEDs use different I2C addresses.

Button works randomly/inconsistently?

1. Use D6 (GPIO12) NOT D3
2. Open Serial Monitor at 115200 baud to see [BTN] debug messages
3. Check wiring: D6 → Button → GND
4. Try a different button - some cheap buttons have poor contacts

Button double-presses?

This is button "bounce". The code has debounce + cooldown. If still bouncing, increase the cooldown value in the code.

WiFi not connecting?

Update ssid and password in the CONFIG section. Check serial monitor at 115200 baud.
