# UI/UX Revamp Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Revamp the UI/UX of `dashboard.html` — replace sidebar with top navbar (desktop) + bottom tab bar (mobile), modernize card/spacing/typography design system — without breaking any existing JS functionality.

**Architecture:** Single-file HTML (`dashboard.html`) with pure CSS design system and vanilla JS. All JS data-binding IDs are preserved untouched. Only CSS `:root` vars, layout HTML structure, and component styling change. One minimal JS patch: add `data-tab` mobile button activation to `switchTab`.

**Tech Stack:** Pure HTML/CSS/JS, Chart.js, Font Awesome 6, Inter + JetBrains Mono fonts, no build tools.

**Spec:** `docs/superpowers/specs/2026-03-17-ui-revamp-design.md`
